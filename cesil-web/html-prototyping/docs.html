Program structure

A CESIL program is made up of two sections, instructions and data.

Instructions

Instruction lines are split into three columns, a label (optional), the operator and an operand (dependent on the operator). Each column is separated by white space.

Comment lines can be added to the instruction set and are started with a (.

The instructions section is terminated with % character on a new line.

Data

The data section starts on a new line after the instruction section terminator.

Data items can be split on multiple lines or by whitespace.

The data section is terminated with a * character on a new line.

Example program

        ( EXAMPLE PROGRAM
                 LOAD     0
        LOOP     STORE    TOTAL
                 IN
                 JINEG    END
                 ADD      TOTAL
                 JUMP     LOOP
        END      PRINT    "THE ANSWER IS "
                 LOAD     TOTAL
                 OUT
                 HALT
        %
        40 2 -1
        *

Instruction reference

Operator Operand
IN Input the next available item of data into the accumulator.
OUT Output the value of the accumulator.
PRINT "text" Output the text in quotes.
LINE Output a new line.
STORE variable Store the value of the accumulator into the given variable.
LOAD variable / literal Load either the value of a variable or a given literal integer into the accumulator.
ADD variable / literal Add either the value of a variable or a given literal integer to the accumulator and overwrite the accumulator with the result.
SUBTRACT variable / literal Subtract either the value of a variable or a given literal integer from the accumulator and overwrite the accumulator with the result.
MULTIPLY variable / literal Multiply the accumulator by either the value of a variable or a given literal integer and overwrite the accumulator with the result.
DIVIDE variable / literal Divide the accumulator by either the value of a variable or a given literal integer and overwrite the accumulator with the result.
JUMP label Jump to an instruction line labeled with the given label.
JIZERO label Jump to an instruction line labeled with the given label if the accumulator is zero.
JINEG label Jump to an instruction line labeled with the given label if the accumulator is negative.
HALT Stop execution

Notes

This version of CESIL has the following differences to the original:

  • The maximum number of instructions that can be executed in a single run is 100. If this is exceeded an error will be returned.
  • Positive integers and zero do not need to be signed.
  • Labels are not limited to six characters.
  • Program header comments are not supported.
